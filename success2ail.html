Install FunnelFlow as a PWA

For the best experience, install FunnelFlow as a Progressive Web App on your device:

iPhone/iPad

  1. Open in Safari
  2. Tap the Share button
  3. Select "Add to Home Screen"

Android

  1. Open in Chrome
  2. Tap the 3-dot menu
  3. Select "Install App"

Desktop

  1. Chrome/Edge: Look for install icon in address bar
  2. Firefox: Menu > "Install"
  3. Safari: File > "Add to Dock"
